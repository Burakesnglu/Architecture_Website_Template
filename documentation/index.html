Installation

Follow these steps to install the Architect Website theme:

Terminal 
# Clone the project
git clone [repo-url]

# Navigate to the project directory
cd architect-website

# Install dependencies
npm install

# Start the development server
npm run dev

Information

Node.js 18.0.0 or higher is required for installation.

Building

To build the project for production, use the following command:

Terminal 
npm run build

The compiled files will be created in the dist directory.

Project Structure

The Architect Website theme has the following folder structure:

architect-website/
├── public/             # Static files
├── src/                # Source code
│   ├── assets/         # Images, fonts, and other media files
│   ├── components/     # Reusable components
│   ├── context/        # React context files
│   ├── hooks/          # Custom React hooks
│   ├── App.jsx         # Main application component
│   ├── App.css         # Main style file
│   ├── index.css       # Global styles
│   └── main.jsx        # Application entry point
├── index.html          # HTML template
├── package.json        # Project dependencies and commands
├── tailwind.config.js  # Tailwind CSS configuration
├── postcss.config.js   # PostCSS configuration
└── vite.config.js      # Vite configuration

Customization

Color Scheme

You can modify the theme's color scheme by editing the tailwind.config.js file:

tailwind.config.js 
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#3B82F6',   // Primary color
        secondary: '#10B981', // Secondary color
        accent: '#8B5CF6',    // Accent color
        // Add other colors here
      },
    },
  },
  // ...
}

Fonts

To change fonts, edit the index.css file or add new fonts from Google Fonts.

Component Styles

Each component has its own style file. You can modify the appearance of components by editing the corresponding CSS files.

Performance

The Architect Website theme is optimized for high performance. The following features ensure fast loading and smooth operation of the theme:

  • Vite - Vite is used for fast development and building
  • Code Compression - Automatic code compression in production builds
  • Image Optimization - Optimized images
  • Code Splitting - Code splitting for faster page loads
  • Lazy Loading - Components and images are loaded as needed

Frequently Asked Questions

To update the theme, pull the latest changes from the GitHub repository and reinstall dependencies:

git pull
npm install

To add a new page, create a new component in the src/components directory and add it to the route configuration in the App.jsx file.

The contact form sends form data to an API. You can customize the form functionality by editing the src/components/ContactForm.jsx file.

To customize Tailwind CSS classes, edit the tailwind.config.js file. Custom colors, fonts, and other design elements can be defined there.
